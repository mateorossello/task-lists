# TaskLists

Practice project - A TUI (Text User Interface) solution for performing CRUD operations on user task lists, built with Rust, Crossterm and Ratatui

---

## Features

-   Manage task lists and tasks directly from the terminal.
-   CRUD operations (Create, Read, Update and Delete).

---

## How to get started

To run the project, you need to have Rust installed.
The application will automatically create a file named lists.json to store your lists and tasks.

---

## Key mapping

-   **Arrow keys or i - j - k - l**
Navigate between lists and tasks.
-   **n / N**
Create a new list or task.
-   **e / E**
Edit selected list or task.
-   **d / D**
Delete selected list or task
-   **Esc**
Cancel current action and return.
-   **Space**
Toggle task completion status.
-   **q**
Quit the application.

---

## Technologies used

-   **Framework:** Crossterm and Ratatui
-   **Language:** Rust
-   **Database:** Local JSON file
